Morse code is a letter encoding system invented in the 1830s and 1840s by Samuel F.B. Morse, who also invented the telegraph. Shown above is a picture of International Morse Code just in English.

To improve efficiency, Morse code was designed so that the length of a letter in Morse code is roughly inverse to the how often it appears. For example, "E" (the most common letter) is just a single dot, or 1 unit, while "Q" (the least common letter) is 13 units long.
